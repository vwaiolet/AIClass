### 정렬

## 선택 정렬
 + O(n^2)
 + 리스트 순회하여 가장 큰 숫자를 끝 인덱스와 교환
 + 마지막 인덱스를 1씩 줄여가며 반복

## 퀵 정렬
 + O(n log n)
 + 리스트 내에서 피벗을 선택
 + 피벗보다 작은 요소들은 모두 피벗의 왼쪽으로
 + 피벗보다 큰 요소들은 모두 피벗의 오른쪽으로

## 힙 정렬
 + O(n log n)
 + 내림차순 정렬
 + 최대/최소 힙을 구성하고 루트와 가장 큰/작은 수와 교환을 반복

## 계수 정렬
 + O(n)
 + 각 숫자가 몇번 등장하는지 카운트
 + 등장 횟수를 누적합
